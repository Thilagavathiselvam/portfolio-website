# 🌸 Thilagavathi Selvam Portfolio Website

Personal Portfolio Website developed using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB and REST APIs to showcase projects, internships, certifications, achievements and technical skills.

## 🚀 Features

* 👩‍💻 Professional Portfolio Website
* 🌙 Dark / Light Theme Toggle
* 📄 Resume Download
* 📊 Portfolio Statistics Dashboard
* 💼 Internship Experience Section
* 📜 Certifications Section
* 📄 Research Paper Presentation
* 🏆 Hackathons & Competitions
* 🎓 Education Details
* 📞 Contact Form
* 🗄 Contact Messages Stored in MongoDB
* 🔗 GitHub & LinkedIn Integration
* 📱 Responsive Design

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/Thilagavathiselvam/portfolio-website.git
```

### Navigate to Backend

```bash
cd backend
```

### Install Dependencies

```bash
npm install
```

### Create .env File

```env
MONGO_URI=mongodb://127.0.0.1:27017/portfolioDB
PORT=5000
```

### Start Backend

```bash
node server.js
```

### Run Frontend

Open:

```text
frontend/index.html
```

using Live Server in VS Code.
